About Me

책임감있는 신입 개발자

대학교 졸업 전까진 웹 개발 쪽에 큰 흥미가 없었지만 졸업 후 국비지원 과정에서 Spring Framework, Mybatis, Jquery 등 여러 프레임워크 및 라이브러리를 사용하면서 웹 개발에 흥미를 느끼게 되어서 웹 개발자가 되기 위해 노력하고 있습니다

저는 제 개발 목표를 '책임감 있는' 개발자로 하고 있습니다. 프로젝트를 하면서 웹 개발은 해야할 일이 많다는 것을 깨달았습니다. 요구사항 분석, UI설계, DB 설계, 기능구현, 유지보수 등등 팀원 각각 책임감을 갖지 않고 개발을 하게 되면 프로젝트가 원활히 진행될 리 만무합니다. 적어도 보수를 받고 개발에 참여하게 된다면, 자기가 맡은 바에 대해 책임을 갖고 최선을 다하는 것은 당연하고 팀원들도 서로 밀고 당기면서 성공적으로 프로젝트를 수행해야 한다고 생각합니다.
